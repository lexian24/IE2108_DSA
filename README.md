# IE2108_DSA Guide

DSA consist of 
### DatA Structure
    Stack and Queue
    Tree 
    Heaps

    Merge Sort
    Heap Sort
### Search algorithm
    Breath first search
    Depth first search
    Greedy
    Dijktra's Algo (shortest path)
    Krustra's Algo (minimum spanning tree)
